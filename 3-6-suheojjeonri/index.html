Readability

  • local variable, constantл“ұ мҪ”л“ң лӮҙмқҳ к°Ғ мҡ”мҶҢл“Өмқ„ мүҪкІҢ нҢҢм•…н•  мҲҳ мһҲлҠ” м •лҸ„мқҙлӢӨ.
  • readabilityк°Җ мўӢмқҖ мҪ”л“ңлҠ” ліҖмҲҳ л“ұмқҳ мң„м№ҳлҘј л°”лЎңл°”лЎң нҢҢм•… к°ҖлҠҘн•ҳлӢӨ.
  • мқјкҙҖм„ұ мһҲлҠ” мҪ”л“ң мҠӨнғҖмқјмқҳ мӮ¬мҡ©мқ„ нҶөн•ҙ readabilityлҘј ліҙмһҘн•  мҲҳ мһҲлӢӨ. (мӨ„л°”кҫёкё°, л“Өм—¬м“°кё° л“ұ)
    • мҪ”л“ң мҠӨнғҖмқјмқҖ м–ём–ҙл§ҲлӢӨ м •н•ҙм§„ мҪ”л“ң мҠӨнғҖмқјмқҙ мһҲлӢӨ. мқҙлҘј мӨҖмҲҳн•ҳл©ҙ лҗңлӢӨ.

Underatandability

  • UnderstandabilityлҠ” мҪ”л“ңмқҳ "м—ӯн• " мқ„ мүҪкІҢ мқҙн•ҙн•  мҲҳ мһҲлҠ” м •лҸ„мқҙлӢӨ.
  • м•„лһҳ мҪ”л“ңм—җм„ң мҪ”л“ңлҘј л”°лЎң л“Өм—¬лӢӨліҙм§Җ м•Ҡкі лҠ” doSomethingмқҙлқјлҠ” н•ЁмҲҳмқҳ кё°лҠҘмқ„ л°”лЎң мқҙн•ҙн•  мҲҳ м—ҶлӢӨ. л”°лқјм„ң м•„лһҳ мҪ”л“ңлҠ” Readableн•ң мҪ”л“ңмқјм§ҖлҠ” лӘ°лқјлҸ„ understandabilityлҠ” мўӢм§Җ м•ҠлӢӨ.
    double[] doSomething(int x1, int x2, int x3){
	double y[] = new double[2];
	double Q = x2*x2 - 4*x1*x3;
	if(Q > 0){
		y[0] = (-x2 + Math.sqrt(Q)) / (2*x1);
		y[1] = (-x2 - Math.sqrt(Q)) / (2*x1);
	} else if(Q == 0)
		y[0] = y[1] = (-x2) / (2*x1);
	return y;
}
