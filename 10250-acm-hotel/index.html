10250 - ACM 호텔

  • 난이도: 브론즈 3
  • 날짜: 2023년 2월 5일
  • 상태: Correct
  • 추가 검토 여부: No

solution

  • 입력된 순서 당 배정될 방의 번호를 계산하는 공식을 만들었다.
  • 공식에 반례가 있어 한번 틀렸다. 예외 처리를 통해 정답 처리 되었다.
  • 솔직히 W값은 필요가 없었는데 왜 입력으로 넣는지가 궁금하다.

code

#include <iostream>
using namespace std;

int main()
{
    int testC;
    int H, W, N;
    cin >> testC;
    int floor;
    int room;
    for(int i = 0; i < testC; i++)
    {
        cin >> H;
        cin >> W;
        cin >> N;
        floor = N%H;
        room = (N / H) + 1;
        if(floor == 0)
        {
            floor = H;
            room--;
        }
        cout << floor * 100 + room << endl;
    }
    return 0;
}
