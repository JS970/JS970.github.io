### Package Manager (homebrew)
- add PATH to .zshrc
```bash
echo 'export PATH=/opt/homebrew/bin:$PATH' >> ~/.zshrc
```
- [homebrew Download](https://brew.sh/index_ko)
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
