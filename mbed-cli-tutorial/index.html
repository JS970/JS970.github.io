Mbed-CLI Tutorial

Mbed-CLI 환경 구축이 완료되었다면 간단한 예제 프로젝트를 진행하면서 사용법을 익혀 보자.

1. 프로젝트 생성

  • 아래 명령어를 입력하면 "project name" 디렉토리가 생성되며 프로젝트 디렉토리로 설정된다. 
    mbed new "project_name" --mbedlib
    • --mbedlib옵션은 Mbed library를 사용하는 Mbed OS 2 프로젝트로 생성하는 옵션이다.
    • STM32F746G-disco Tutorial은 Mbed library를 사용하므로 이 옵션으로 생성해야 한다.
    • 이외의 자세한 명령어 및 옵션은 Mbed CLI 1 Create에서 정보를 얻을 수 있다.
  • 라이브러리를 추가할 때는 아래의 명령어를 사용하면 된다.
    mbed add http://os.mbed.com/teams/ST/code/LCD_DISCO_F746NG/
    • mbed add뒤에 라이브러리를 입력하면 된다.
    • 위에서 추가한 라이브러리는 STM32F746G-disco Tutorial에서 사용되는 LCD 라이브러리이다.
